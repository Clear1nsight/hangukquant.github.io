Notes

execrates, commrates, longswps, shortswps are presented in decimals. execrates = [0.001, 0.005, ...] encodes that 0.1% of notional value transacted is deducted as execution costs. commrates specify commisions in the same units (as percentage of notional value), as for overnight swap rates for both long and short positions.

hypothesis_tests(num_decision_shuffles=1000, num_data_shuffles=10) async

Conducts monte carlo permutation p-value hypothesis tests on the performance of the trading strategy represented by BaseAlpha object.

Parameters:

Name Type Description Default
num_decision_shuffles int

Number of decision shuffles for monto carlo permutation tests. Default is 1000.

 1000
num_data_shuffles int

Number of data shuffles for permutation tests. Default is 10.

 10

Returns:

Name Type Description
dict

A dictionary containing the results of hypothesis tests. Keys: - 'timer_p': p-value from asset-timing test shuffling time-series asset returns. - 'picker_p': p-value from asset-picking shuffler test shuffling cross-sectional asset returns. - 'trader_p1': p-value from decision-making test shuffling time-series and cross-sectional asset returns. - 'trader_p2': p-value from decision-making test obtained from shuffling market data.
